# @vuepress/plugin-medium-zoom

> medium-zoom plugin for vuepress

## Install

```bash
yarn add -D @vuepress/plugin-medium-zoom
# OR npm install -D @vuepress/plugin-medium-zoom
```

## Usage

```javascript
module.exports = {
  plugins: ['@vuepress/medium-zoom'] 
}
```

## Options

### selector

- Type: `string`
- Default: `.content img`
