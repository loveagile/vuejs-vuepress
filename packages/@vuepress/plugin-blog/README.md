# @vuepress/plugin-blog

> theme plugin for vuepress

## Usage

```javascript
module.exports = {
  plugins: ['@vuepress/blog'] 
}
```

## Options

### pageEnhancers

- Type: `function`
- Default: `/i18n/`

### categoryIndexPageUrl

- Type: `string`
- Default: `/category/`

### tagIndexPageUrl

- Type: `string`
- Default: `/tag/`
